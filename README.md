## console

console is a small command-line interface tool for [undertasker](https://github.com/braycarlson/undertasker)

## Download

- [Installer](https://github.com/braycarlson/undertasker/releases/download/0.1/installer.exe)
- [Portable](https://github.com/braycarlson/undertasker/releases/download/0.1/portable.zip)

## Getting Started
1. Download and install the undertasker [installer](https://github.com/braycarlson/undertasker/releases/download/0.1/installer.exe)
2. Use `console.exe` to add command(s) to `command.txt` or manually add a path to an executable and/or a terminal command to `command.txt`.
3. Run `undertasker.exe`.

## Note
1. `command.txt` **must** remain in the same directory as `undertasker`.
2. Each command in `command.txt` **must** be on a separate line.

## Options

### **Start: Undertasker**
Open undertasker and run every command in `command.txt`

### **Search: File**
Search all drives for a file by file name and extension

### **Search: Folder**
Search all drives for a file by folder name and extension

### **Search: Microsoft Software**
Search for Microsoft software

### **Search: Microsoft Settings**
Search for Microsoft settings

### **Command: Open**
Open `command.txt` in your default text editor

### **Command: Location**
Open the location of `command.txt` in File Explorer

### **Command: View**
Print every command in `command.txt` to the terminal

### **Command: Clear**
Clear the `command.txt` file

### **Command: Remove**
Select command(s) to remove from `command.txt`

### **Exit**
Close `console`
